Slide 4 · V3 Table-Centric LLM + Soft Rules

Algorithm (New)

  • Chunk by full table (title/context, headers, rows), not by isolated rows.
  • Prompt includes soft rules: field synonyms, preferred table type, column/year semantics, unit handling.
  • LLM returns structured evidence: value, table_id, row_label, column_label, unit, confidence.
  • Arbitration layer keeps sanity checks and fallback behavior.

Key Result

Exact Accuracy: 100.0% (30/30) · MAPE: 0.0%

Major gain came from exposing structural context (headers + table role) and codifying domain heuristics as soft rules.
